# Archive — superseded material (kept for reference, not in active use)

These files belong to the project's earlier rainfall-runoff approach, which was
replaced (on supervisor instruction) by the discharge-only statistical (ARIMA)
approach. Nothing here is used by the current pipeline, app, or thesis.

| Item | Why it is archived |
|------|--------------------|
| `PROJECT_CONTEXT.md` | The old master spec describing the two-bucket rainfall-runoff model, the Ogun-Osun (Nigeria) basin, and parameter transfer. Superseded by the ARIMA approach. |
| `POWER_Point_Daily_19900101_20201231_007d50N_003d50E_LST.csv` | NASA POWER rainfall/temperature data for the Ogun-Osun basin. No longer used — the model uses observed discharge only. |
| `logs/` | Transient run logs from pipeline and Streamlit boots. |

Safe to delete entirely if disk space is needed.
